# 实践指南

本目录包含 Kosong 在实际应用场景中的完整示例和最佳实践。

## 📝 指南列表

### [多轮对话](./multi-turn-conversation.md)
学习如何维护对话历史、管理上下文，以及构建完整的聊天机器人应用。

**适用场景：** 聊天机器人、对话式 AI 助手、客服系统

### [流式输出](./streaming-output.md)
掌握流式输出的工作原理，实现实时响应显示和流式工具调用处理。

**适用场景：** 实时对话界面、渐进式内容生成、用户体验优化

### [自定义工具](./custom-tools.md)
学习如何创建自定义工具，包括参数验证、错误处理和异步实现。

**适用场景：** 扩展 AI 能力、集成外部 API、实现特定业务逻辑

### [错误处理](./error-handling.md)
了解常见异常类型和处理策略，实现健壮的错误处理和优雅降级。

**适用场景：** 生产环境部署、提高应用稳定性、用户体验优化

### [生产部署](./production-deployment.md)
获取性能优化建议、并发处理策略、日志监控和成本控制方案。

**适用场景：** 生产环境部署、性能优化、运维监控

## 🎯 学习路径建议

1. **初学者：** 多轮对话 → 流式输出 → 错误处理
2. **进阶开发者：** 自定义工具 → 生产部署
3. **架构师：** 所有指南 + [架构设计文档](../architecture.md) + [高级主题](../advanced/README.md)
